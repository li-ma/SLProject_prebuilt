How to build OpenCV with Contributions on Windows with VisualStudio:
- cd into a directory where you clone opencv and opencv_contrib
- Clone OpenCV with: git clone https://github.com/Itseez/opencv.git
- Clone OpenCV Contributions with: git clone https://github.com/Itseez/opencv_contrib.git
- Open CMake GUI Application.
  - Set the source code path to the top-level opencv folder
  - Set the build path to .../opencv/build (it will create the folder)
  - Press Configure
  - Type "extra" in the search field to find OPENCV_EXTRA_MODULES_PATH and set it to the opencv_contrib/modules folder
  - Check flag WITH_FFMPEG
  - Uncheck flag WITH_VTK
  - Press Generate
- Go to the build folder and open the opencv.sln solution file in Visual Studio
  - Set the configuration to Debug/x64
    - Build the ALL_BUILD project
    - After successful build, build CMakeTargets/INSTALL
    - Copy all from opencv\build\install\x64\vc??\ into SLProject\_lib\prebuilt\OpenCV\x64\vc??\
  - Set the configuration to Release/x64
    - Build the ALL_BUILD project
    - After successful build, build CMakeTargets/INSTALL
    - Copy all from opencv\build\install\x64\vc??\ into SLProject\_lib\prebuilt\OpenCV\x64\vc??\
	
- Copy all files from the folder opencv\build\install\include\ to SLProject/lib_SLExternal/opencv/include
